🎮 About Me

I’ve been developing games since the Flash and GameMaker days. My journey began with RPG paper-based games I created to play with my younger sister. Now, I work solo using Godot, focusing on game systems, design, and code.

Inspired by studios like Ankama and Motion Twin productions, I aim to release on Steam, Itch.io, and more.

🧪 Current Project

I'm building an untitled 2D isometric action RPG inspired by Diablo and Path of Exile. Development started in March 2025.
